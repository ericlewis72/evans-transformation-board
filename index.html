Type
Definition
Customer
Full customer transfer — all sub-processes moving to E3. Clean UX-only migrations.
Process
A specific workflow or process being transformed, not the whole customer. Used when integration remains untouched and only a UI or functional process is moving.
Integration
Semi-integrated customers (flat-file, sFTP, EDI, Legacy WS, Windows Service, etc.) where the integration path itself is part of the transformation.
